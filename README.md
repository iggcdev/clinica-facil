# Clínica Fácil Frontend Prototype

Este repositório contém um protótipo simples de frontend para uma clínica.
O sistema não possui backend real e utiliza apenas **LocalStorage** para persistir dados no navegador.

## Funcionalidades
- Cadastro e login de pacientes
- Visualização de especialidades e médicos disponíveis
- Agendamento e cancelamento de consultas
- Histórico de consultas
- Notificações de lembrete de consultas (alertas simples)
- Área administrativa para médicos visualizarem sua agenda

## Como usar
1. Clone o repositório ou baixe os arquivos.
2. Abra `index.html` em um navegador moderno.
3. Utilize o formulário de cadastro para criar uma conta de paciente ou faça login com um dos médicos pré-cadastrados:
   - `ana@clinica.com` / senha `1234`
   - `bruno@clinica.com` / senha `1234`
   - `carla@clinica.com` / senha `1234`

Todo o fluxo acontece localmente no navegador e pode ser reiniciado limpando o armazenamento local.
